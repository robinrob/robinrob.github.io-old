Previous Projects

Some previous projects I have worked on.

  • Commercial Salesforce/Google synchronisation application

    I was a core developer on a Force.com application to synchronise Email, Calendar and Contact information between Google Apps and Salesforce.com platforms. The app made use of advanced techniques such as scheduled and batch Apex, and used triggers and web services. I architected and built a second stand-alone Force.com application acting as a licensing server, to perform product verification and registration, offering trial and paid functionality for the business' Force.com products.

  • Salesforce-AWS Email Backup and Archiving System

    I designed and implemented an archiving and retrieval system for Salesforce Email Messages, using Python and running on Amazon EC2. Business-critical Salesforce data was stored and synced onto Amazon Web Services S3, reducing data storage costs. The system was successfully deployed for two separate customers. Automated testing helped ensure a high reliability of the system.

  • Heroku Database Management Tool

    I wrote a tool to perform backups of Heroku application databases, using Ruby and running on Amazon EC2, with customisable backup rotation functionality.
