RSS Reader Script

Short Ruby script to open RSS links using terminal interface

I wanted a quick, lightweight way to browse and open articles from RSS feeds from within the terminal. I wrote a short Source code repository for RSS Reader Script project for quickly opening RSS feed articles in a web browser.
